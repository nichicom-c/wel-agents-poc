# Security Policy (excerpt)

Synthetic sample document for the RAG PoC. Not real WEL-MOTHER content.

## Credentials

Never commit credentials, private keys, or access tokens to source control. Use
a secrets manager or environment variables. Rotate long-lived credentials at
least every 90 days.

## Data classification

- **Public**: May be shared externally.
- **Internal**: Default for business data; do not share outside the company.
- **Confidential**: Restricted; access on a need-to-know basis only.

## Access control

Apply least privilege. Production access requires multi-factor authentication
and is reviewed quarterly.

## Incident reporting

Report suspected security incidents immediately via the #security channel and
the on-call hotline.
